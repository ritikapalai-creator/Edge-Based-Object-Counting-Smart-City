# Edge-Based Object Counting System for Smart City Applications

## Overview
This repository contains the MATLAB implementation of the research paper:

EDGE-BASED OBJECT COUNTING SYSTEM FOR SMART CITY APPLICATIONS: AN ADAPTIVE MORPHOLOGICAL SEGMENTATION FRAMEWORK

## Features
- Edge Detection
- Morphological Segmentation
- Object Counting
- Image Processing

## Technologies
- MATLAB
- Image Processing Toolbox

## Publication Status
Accepted for publication in IJISRT.
